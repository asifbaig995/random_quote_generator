# Random Quote Generator

## About the Project
This is a beginner-friendly web project built using HTML, CSS, and JavaScript. The application displays motivational quotes and allows users to generate a new quote by clicking a button.

## Features
- Display motivational quotes
- Generate a new quote with one click
- Copy quotes to the clipboard
- Simple and attractive user interface

## Technologies Used
- HTML
- CSS
- JavaScript

## What I Learned
- DOM manipulation
- Event handling in JavaScript
- Working with buttons and user interactions
- Updating webpage content dynamically
- Basic webpage styling using CSS
